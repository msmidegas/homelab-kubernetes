# Homelab Kubernetes Project

## Goal
Build a production-like Kubernetes environment for learning DevOps.

## What I will implement
- Linux server (Ubuntu)
- Docker containers
- Kubernetes cluster (k3s)
- GitOps deployment (Argo CD)
- Monitoring (Prometheus + Grafana)

## Architecture (future state)
GitHub → CI/CD → Argo CD → Kubernetes → Apps → Monitoring
